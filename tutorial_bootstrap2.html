Urban Parks tutorial

Hi! This is a demo application that shows how you can do geo-coding problems with web mapping tools using the PyBossa framework in Crowdcrafting.org.

The application is really simple. It loads a city from the world into the web map, centers the map and the goal is to browse the map and find one urban park.

The web map provides different layers so you can change between them in order to get some help from satellite images.

When you find an Urban Park, click in the black button: Urban Park Marker and place it in the middle of the park. Then click submit and you will have completed one task.
